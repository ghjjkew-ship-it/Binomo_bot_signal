# Binomo Signals Bot

Телеграм-бот сигналов для пары AUD/CAD (OTC).

Команды:

- /start — приветствие и инструкция
- /новый — получить новый торговый сигнал
- /статистика — посмотреть статистику точности

Использует TwelveData (основной) и yfinance (фоллбек).
